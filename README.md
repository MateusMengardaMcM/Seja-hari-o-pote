# Seja hari, o pote

Projeto voltado para fins educacionais, com intenção de aprender javascript, focando nos conceitos das arrays e matrizes, alem das funcionalidades da condicional if else.

## Aprendizados

Compreendi os conceitos das arrays, das matrizes e da condicional if else.

## Funcionalidades

- usar os botões para escolher sua trajetória e definir seu final.

## Stacks utilizada

**Front-end:** Html, css & JavaScript.



## Deploy

Para fazer o deploy desse projeto faça um clone do repositório e execute o arquivo em index.html.

```bash
  git clone https://github.com/MateusMengardaMcM/Seja-hari-o-pote
```
